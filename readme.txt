Citationic help your visitors to cite your articles using correct styles: APA, MLA, Harvard, Vancouver, Chicago, IEEE citation guides.

== Description ==

Simple, lighweight multi-styles citation plugin. Only CSS, no JS required! Help your visitors cite your texts: APA, MLA, Harvard, Vancouver, Chicago, IEEE citation guides. Insert shortcode [citationic] in your texts.

= Key Features =

*	Simple & lightweight
*	No JS required, tabs made by CSS
*	Works in all major browsers
*	Just use shortode [citationic] 

= Demo =

*	[Click here](https://essayzz.com/essays/hyperloop-transportation-system/) and look at the right sidebar

= Languages =

Only English language supported.

== Installation ==

1. Upload the `citationic` folder to the `/wp-content/plugins/` directory
1. Activate the Cite plugin through the WordPress 'Plugins' dashboard.
1. Add to any page using shortcode `[citationic]`
